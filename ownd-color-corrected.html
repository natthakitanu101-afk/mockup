CTA Button #3DB140 บน White ไม่ผ่าน WCAG

ปุ่ม CTA หลัก (เช่น "ลงทะเบียนเลย") ใช้ #3DB140 บนพื้นขาว contrast ratio = 2.8:1 ❌ ต้องการ 4.5:1 — แก้โดยใช้ #00704C (Green-500) สำหรับ CTA บน white และใช้ #3DB140 เฉพาะบน dark bg เท่านั้น

+

Warm Neutral #FAF3ED ไม่มีใน System

หน้า Matching ใช้พื้นหลังสี beige/cream (#FAF3ED) ในส่วน "ทำไมเช่าเพื่อซื้อกับ OWND" แต่ไม่มีใน Color System เดิม — เพิ่มเป็น warm-50 token

✓

Dark Background = #083538 ไม่ใช่ #0A2633

Section สีเข้ม (Nav, Plan cards, Article, Footer) ทุกจุดใช้ #083538 เป็นหลัก ไม่ใช่ #0A2633 ที่เคยตั้งเป็น Green-900 — ปรับ token ให้ตรง

✓

Tab Active / Card Green = #026348

Tab Buyer/Investor active state และ Investment card background ใช้ #026348–#016548 ซึ่งอยู่ระหว่าง Green-500 กับ Green-600 — จัดเข้า token Green-600
