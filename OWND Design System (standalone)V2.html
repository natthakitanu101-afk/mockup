Accessibility

Resource · WCAG 2.1 AA

เป้าหมาย WCAG 2.1 AA — ตัวหนังสือ contrast ≥ 4.5:1 (ปกติ) / 3:1 (ตัวใหญ่). ค่าด้านล่างมาจากการวัด pixel จริง (อ้างอิงงาน UI audit เดิม) กฎสำคัญ: lime #79F233 และ #3DB140 ใช้เป็นตัวหนังสือ/CTA ได้เฉพาะบนพื้นเข้มเท่านั้น — บนพื้นขาวไม่ผ่าน

Design Tokens (export)

Resource · for Dev

Single source of truth — token ทั้งหมดถูก export เป็น 3 รูปแบบให้ dev เลือก import ตาม stack วางไว้ที่โฟลเดอร์ ownd-ds-assets/ ข้างไฟล์นี้

⬇ ownd-tokens.css — CSS custom properties ⬇ ownd-tokens.json — Style-Dictionary / Figma Tokens ⬇ ownd.tailwind-preset.js — Tailwind preset
วิธีใช้
import
/* 1) CSS — import แล้วใช้ var() ได้เลย */
@import "ownd-ds-assets/ownd-tokens.css";
.btn{ background: var(--ownd-green-500); border-radius: var(--ownd-radius-pill); }

/* 2) Tailwind — ใส่ใน tailwind.config.js */
module.exports = { presets: [require('./ownd-ds-assets/ownd.tailwind-preset.js')] }
/* → bg-brand-500, text-ink, rounded-pill, shadow-card, duration-base ... */

/* 3) JSON — feed เข้า Style Dictionary / Figma Tokens / build script */
